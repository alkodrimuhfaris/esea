## Requirements
1. <a href="https://nodejs.org/en/download/">Node JS</a>
2. Node_modules
3. Physical device or Android emulator
4. Code editor (Visual Studio Code)
5. Web Browser (Google Chrome)

## How to run the app?
1. Open app's directory in CMD
2. Type `yarn install`
3. Type `yarn start`
